# How to install and run

1. Open your favorite IDE (like Visual Studio Code or WebStorm) and type in terminal ```git clone https://github.com/artyom-n/sonarworks.git```
2. When you are in the project folder, type in terminal ```npm install``` to install all dependencies
2. After installation type in terminal ```npm start``` to run the project
4. In the browser type ```http://localhost:3000/```
