# Sonarworks

Project demo is available here: http://artyom-n.github.io/sonarworks

## Guidelines of how to run it

1. Open Visual Studio Code or another IDE and type in terminal ```git clone https://github.com/artyom-n/sonarworks.git```
2. When you are in the project folder, type in terminal ```npm install``` to install all dependencies
2. After installation type in terminal ```npm start``` to run the project
4. In the browser type ```http://localhost:3000/```
